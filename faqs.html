Why isn’t the BlueJay DIY Application Free?

  • OEM programs like GEF12Host or FJ Programming Tool are not actually free — they are licensed software that have been pirated and redistributed on unofficial sites.
  • BlueJay is a fully supported multi-controller application offering features OEM tools lack, such as:
    • Save / Restore
    • Profiles
    • User Guide
    • Clear documentation for all common tuning parameters
    • Email support, bug fixes, and continuous updates
